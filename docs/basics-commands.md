# Basic Docker Commands

## Getting Started

```
$ docker help
```

## Working with Containers

```
$ docker run
```

```
$ docker ps
```

```
$ docker stop
```

```
$ docker start
```

```
$ docker rm
```

## Working with images

```
$ docker images
```

```
$ docker pull
```

```
$ docker rmi
```

```
$ docker tag
```

```
$ docker save
```

```
$ docker load
```

## SSH?

```
$ docker run -d -p 3306:3306 --name mariadb mariadb:10.4.4
```

```
$ docker exec
```

## Operations

```
$ docker logs
```

```
$ docker stats
```
